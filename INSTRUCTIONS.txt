
            CROSSWORD GENERATOR — Quick Start
            ==================================

  SETUP (one time)
  ────────────────
  1. Install Python from https://www.python.org/downloads/
     IMPORTANT: check "Add Python to PATH" during installation.

  2. Double-click  setup.bat  and wait for "Setup completed!"


  USAGE
  ─────
  Double-click  run.bat  and choose:

    [1] From Excel file — you provide words and clues
    [2] Auto-generate   — uses built-in word bank

  Or drag your .xlsx file directly onto run.bat.

  Results appear in the "output" folder (PDF, Excel, SVG).


  COMMAND LINE
  ────────────
  Open Command Prompt: press Win+R, type cmd, press Enter.
  Navigate to the program folder:  cd C:\path\to\crossword

  From Excel file:
    run.bat  myfile.xlsx
    run.bat  myfile.xlsx  --title "My Puzzle"
    run.bat  myfile.xlsx  --symmetry
    run.bat  myfile.xlsx  --grid-size 17

  Auto-generate:
    run.bat  --generate
    run.bat  --generate  --title "Daily Puzzle"
    run.bat  --generate  --seed 42


  EXCEL FORMAT (for mode 1)
  ─────────────────────────
  3 columns: Number | Clue | Answer

     A       B                        C
     1       Man's best friend        DOG
     2       Capital of France        PARIS
     3       Frozen water             ICE

  Rows 1-2 are headers, data from row 3.
  Answers: UPPERCASE, letters only, 3-15 characters.
  See input_example.xlsx for reference.
